# Teaching Notes

## User Preferences

- Wants questions visible with answers hidden (guess first, then reveal)
- Wants a final puzzle exam to test applied knowledge
- Using TypeScript as the pilot — will extend to other topics if successful
- Focus is interview preparation, not general learning

## Lesson Structure

Each lesson covers 4-6 related questions grouped by concept (not just difficulty).
Order: explanation → interactive Q&A cards → mini-quiz → next lesson link.

## Progression

Lessons build on each other:
1. Fundamentals → 2. Defining Types → 3. Generics → 4. Special Types →
5. Functions & Properties → 6. Narrowing & Guards → 7. Utility Types →
8. Advanced Features → 9. TS Patterns → 10. Advanced TS → 11. Final Exam
