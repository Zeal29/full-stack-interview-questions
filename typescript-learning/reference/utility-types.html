Cheat Sheet

TypeScript Utility Types

Every built-in utility type — syntax, what it does, and a practical example.

Object Transforms

Modify the shape of object types — make things optional, required, readonly, or pick specific properties.

UtilityDescriptionExample
Partial<T> Makes all properties optional 
interface User { name: string; age: number; }
type PartialUser = Partial<User>;
// { name?: string; age?: number }

function update(id: number, data: Partial<User>) {
  // only send changed fields
}
Required<T> Makes all properties required 
interface Config { host?: string; port?: number; }
type StrictConfig = Required<Config>;
// { host: string; port: number }
Readonly<T> Makes all properties readonly 
type Frozen = Readonly<{ x: number; y: number }>;
const point: Frozen = { x: 0, y: 0 };
// point.x = 1;  // Error!
Pick<T, K> Extract a subset of properties 
interface User {
  id: number; name: string; email: string;
}
type UserPreview = Pick<User, "name" | "email">;
// { name: string; email: string }
Omit<T, K> Remove a subset of properties 
interface User {
  id: number; name: string; password: string;
}
type SafeUser = Omit<User, "password">;
// { id: number; name: string }
Record<K, V> Object type with keys K, values V 
type Role = "admin" | "editor" | "viewer";
type Perms = Record<Role, boolean>;
// { admin: boolean; editor: boolean; viewer: boolean }

const perms: Perms = {
  admin: true, editor: true, viewer: false,
};

Union & Nullability

Filter, exclude, and narrow union types.

UtilityDescriptionExample
Exclude<U, E> Remove types E from union U 
type T = string | number | boolean;
type NoBool = Exclude<T, boolean>;
// string | number
Extract<U, E> Keep only types matching E from U 
type T = string | number | boolean;
type OnlyStr = Extract<T, string>;
// string
NonNullable<T> Remove null and undefined from T 
type Maybe = string | null | undefined;
type Sure = NonNullable<Maybe>;
// string

Function & Class Types

Extract metadata from function and constructor signatures.

UtilityDescriptionExample
ReturnType<T> Extract return type of a function 
function getUser() {
  return { name: "Alice", age: 30 };
}
type User = ReturnType<typeof getUser>;
// { name: string; age: number }
Parameters<T> Extract param types as a tuple 
function createUser(name: string, age: number) {}
type Args = Parameters<typeof createUser>;
// [string, number]
InstanceType<T> Extract instance type of a constructor 
class Box {
  constructor(public value: number) {}
}
type BoxInstance = InstanceType<typeof Box>;
// { value: number }
ThisParameterType<T> Extract the this parameter type 
function greet(this: { name: string }) {
  return `Hi, ${this.name}`;
}
type ThisType = ThisParameterType<typeof greet>;
// { name: string }
OmitThisParameter<T> Remove the this parameter from type 
type FreeFn = OmitThisParameter<typeof greet>;
// () => string (no `this` needed)

String Case Transforms

Convert string literal types between cases.

UtilityDescriptionExample
Uppercase<S> Uppercase entire string 
type Upper = Uppercase<"hello">;
// "HELLO"
Lowercase<S> Lowercase entire string 
type Lower = Lowercase<"HELLO">;
// "hello"
Capitalize<S> Capitalize first character 
type Cap = Capitalize<"hello world">;
// "Hello world"
Uncapitalize<S> Lowercase first character 
type Uncap = Uncapitalize<"HelloWorld">;
// "helloWorld"

Common Compositions

Real-world patterns that combine multiple utility types.

PatternWhat It DoesExample
Partial<Pick<T, K>> Make specific props optional 
interface User {
  id: number; name: string; email: string;
}
type Patch = Partial<Pick<User, "name" | "email">>;
// { name?: string; email?: string }
Omit<T, keyof U> Remove all keys that exist on U 
interface Base { id: number; created: Date; }
interface Full extends Base { name: string; }
type NoBase = Omit<Full, keyof Base>;
// { name: string }
Required<Pick<T, K>> Force specific props to be required 
interface Config {
  host?: string; port?: number; debug?: boolean;
}
type Prod = Required<Pick<Config, "host" | "port">>;
// { host: string; port: number }
Readonly<Omit<T, K>> Immutable with excluded keys 
interface State {
  id: number; data: string; _internal: any;
}
type PublicState = Readonly<Omit<State, "_internal">>;
// { readonly id: number; readonly data: string }
Record<keyof T, V> Remap object values to a new type 
type Flags<T> = Record<keyof T, boolean>;
interface Perms { read: string; write: string; }
type PermFlags = Flags<Perms>;
// { read: boolean; write: boolean }
