# Mission: Master TypeScript for Technical Interviews

## Why

Preparing for backend and full-stack technical interviews. TypeScript is a critical skill — most modern companies expect strong TS knowledge including type system internals, utility types, generics, and advanced patterns.

## Goal

Achieve fluency in TypeScript's type system such that any interview question on types, generics, narrowing, or advanced patterns can be answered confidently and correctly. Go beyond memorization to genuine understanding and application.

## Success Criteria

- Can explain any TypeScript concept from first principles (not just recite)
- Can predict type checker output for unfamiliar code
- Can write type-safe APIs from scratch using generics, unions, and utility types
- Can debug type errors quickly and fix them without `any`
- Score 80%+ on the final puzzle exam

## Scope

50 TypeScript interview questions organized into 11 progressive lessons, from fundamentals through type-level programming. Includes interactive hidden-answer cards, mini-quizzes, and a comprehensive puzzle-based final exam.

## Timeline

Multi-session. One lesson per session recommended, with the final exam as a capstone.
