# Mission: Master React for Technical Interviews

## Why

Preparing for frontend and full-stack technical interviews. React is the most in-demand frontend framework. Need deep understanding of hooks, rendering behavior, performance, and modern React (18/19) features — not just surface API knowledge. Interviewers probe the internals: why re-renders happen, how reconciliation works, how to debug effect timing and stale closures, and when to reach for which optimization.

## Goal

Achieve fluency such that any React interview question on hooks, rendering, performance, patterns, or internals can be answered from first principles and applied to real debugging scenarios. Go beyond memorizing API signatures to understanding the render cycle, the reconciler, and the trade-offs of every state-management choice.

## Success Criteria

- Can explain React's render cycle from first principles (render → reconcile → commit → effect)
- Can debug stale closures, infinite re-render loops, and effect timing issues
- Can choose the right state management approach for any scenario (useState vs useReducer vs Context vs external store)
- Can optimize re-renders with profiling-driven decisions (React.memo, useMemo, useCallback — and knowing when NOT to)
- Scores 80%+ on the final puzzle exam

## Scope

50 React interview questions organized into 9 progressive lessons plus a comprehensive 15-puzzle final exam. Includes interactive hidden-answer cards, mini-quizzes, and applied knowledge puzzles covering hooks, rendering, performance, patterns, and React 19 internals.

## Timeline

Multi-session. One lesson per session recommended, with the final exam as a capstone.
