## How to Use This Workspace

1. Open each lesson file in a browser (just double-click the `.html`).
2. Read the question, form your answer mentally, then click **"Reveal Answer"**.
3. Take the mini-quiz at the end of each lesson.
4. Finish with the **Final Exam** — 15 puzzles from easy to hard.
5. Keep notes in `learning-records/`.
