# Teaching Notes

## User Preferences

- Wants questions visible with answers hidden (guess first, then reveal)
- Wants a final puzzle exam to test applied knowledge
- Focus is interview preparation, not general learning
- React is the second topic after TypeScript (which was the pilot)

## Lesson Structure

Each lesson covers 3-10 related questions grouped by concept (not just difficulty).
Order: conceptual teaching → interactive Q&A cards (guess then reveal) → mini-quiz → ask reminder → next lesson link.

## Progression

Lessons build on each other:
1. Fundamentals → 2. State & Core Hooks → 3. Forms, Lists & Rendering →
4. Refs, Events & Memoization → 5. State Management & Context →
6. Component Patterns → 7. Custom Hooks & Advanced APIs →
8. Performance & Concurrent React → 9. Ecosystem, SSR & React 19 →
10. Final Exam (15 puzzles)

The first four lessons cover the "must-know" basics. Lessons 5-7 add patterns and
custom abstractions. Lesson 8 dives into performance and the concurrent renderer —
the differentiator for senior interviews. Lesson 9 covers modern React (Server
Components, React 19) which is increasingly asked. The final exam tests applied
debugging and design skill.
